# Countdown App

Family countdown widget app built with React Native + Expo.

Count down to vacations, birthdays, holidays — with beautiful gradient cards and home screen widgets.

## Stack
- React Native + Expo SDK 52
- expo-router (file-based navigation)
- AsyncStorage (local persistence)
- RevenueCat (in-app purchases)

## Features
- Create countdowns with emoji, color themes, and custom photos
- Live ticker (days / hours / minutes / seconds)
- Annual recurring events
- PRO unlock ($2.99 one-time) — unlimited events, all themes, photo backgrounds
- Tip jar (☕ $1 / 🧁 $3 / 🍕 $5)

## Roadmap
- V1.0: Core app (MVP)
- V1.1: Home screen + lock screen widgets, Apple Watch, push notifications

## Getting Started
```bash
npx expo start
```
